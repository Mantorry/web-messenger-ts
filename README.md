**Node version 14.x**

### Склонировать репозиторий

```shell
git clone https://github.com/AntonioErdeljac/next13-messenger.git
```

### Установка пакетов

```shell
npm i
```

### .env


```js
DATABASE_URL=
NEXTAUTH_SECRET=

NEXT_PUBLIC_PUSHER_APP_KEY=
PUSHER_APP_ID=
PUSHER_SECRET=

NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=

GITHUB_ID=
GITHUB_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

### Запуск конфига Prisma

```shell
npx prisma db push

```

### Запуск программы

```shell
npm run dev
```
